# webs-wc2026-notes

本仓库用于归档与发布多个独立的 HTML 页面。这些页面以笔记、工具或信息页的形式存在，彼此之间没有强关联，也不针对任何特定域名或网站。

## 项目简介

本仓库的主要目的是整理和发布一系列自包含的 HTML 文件。每个文件通常对应一个独立的功能或主题，可以直接在浏览器中打开使用。仓库中的内容不构成一个完整的网站或应用，而是作为零散的资源存放点。

## 目录说明

```
webs-wc2026-notes/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- **pages/**：每个 `.html` 文件都是一个独立的页面，可以直接访问。
- **assets/**：如果页面需要额外的样式、脚本或图片，会统一放在此目录下，按需组织。

## 页面归档说明

- 每个页面在创建时尽量保持自包含，减少外部依赖。
- 页面之间可能共享部分资源，但原则上各自独立运行。
- 归档的页面可能处于不同完成度：有的作为笔记草稿，有的功能完整，有的仅供演示。
- 不保证所有页面会长期维护或更新。

## 维护说明

- 本仓库不定期更新，没有固定的发布周期。
- 页面内容可能随时修改、移除或替换，恕不另行通知。
- 如果你发现某个页面无法正常使用，可能是资源路径或依赖发生了变化，请自行检查或忽略。
- 欢迎通过 Issues 提出建议或报告问题，但请注意：本仓库不提供任何使用担保或技术支持。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 目录下的任意 `.html` 文件，即可在浏览器中查看。
3. 如果需要部署到服务器，请确保 `assets/` 目录与页面文件保持相对路径一致。
